#Silver
Simple programming language geared to teach<br />
programming logic but still get people typing...

#Installation
1. Open the command line...
2. Type: `cd /usr/local`
3. Type: `git clone https://github.com/blubrackets/silver-lang`
4. Sip some coffee...
5. Type: `./silver-lang/install`

######And your done!!
